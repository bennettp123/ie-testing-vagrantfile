# IE testing using Vagrant

Shamelessly ripped from https://gist.github.com/anthonysterling/7cb85670b36821122a4a

Usage:
```
IE=<version> vagrant up
```

Where `<version>` is one of the following:
* `"ie6.xp"`
* `"ie7.vista"`
* `"ie8.xp"`
* `"ie8.win7"`
* `"ie9.win7"`
* `"ie10.win7"`
* `"ie10.win8"`
* `"ie11.win7"`
* `"ie11.win81"`
* `"msedge.win10"`
